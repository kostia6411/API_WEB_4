# Музыкальный плейлист

Данная программа позаоляет несколькими способами получать информацию о песнях (Дата выхода, музыкант название, ссылки на текст и изображение). А также скачивать обложки и текст песен. Для работы этой программы используеться данные с сайта  [genius.com](https://genius.com/). 

+ search.py - позволяет найти песню или автора через навание песни или имя автора. 
+ songs.py - позволяет найти песню по id.
+ artists.py - позволяет найти автора по id.

## Установка зависимостей

Python3 должен быть уже установлен. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Переменная окружения 

Создайте файл .env в корневой директории проекта и добавьте туда переменную API_KEY, содержащую API ключ с сайта [genius.com](https://genius.com/). Зарегистрируйтесь на genius.com и в своем профиле найдите ваш API ключ.

Как выглядит .env:

`API_KEY = "{ВАШ КЛЮЧ}"`

## Запуск программы

Чтобы запустить search.py:

```
python search.py --name {Название песни или имя исполнителя}
```

Чтобы запустить songs.py:

```
python songs.py --song_id {id песни}
```

Чтобы запустить artists.py:

```
python artists.py --artist_id {id исполнителя}
```
